# CoreWatch 5.7.0 구현 상태

## 현재 버전

- 앱 및 설치 프로그램: `5.7.0`
- 이번 기능: 메모리 상세 분석

## 5.7.0 구현 내용

- 물리 메모리 사용·가용량, 시스템 캐시를 표시합니다.
- 현재·한계·최고 커밋과 사용률을 계산합니다.
- Paged/Non-paged 커널 풀을 구분합니다.
- 페이지 파일 자동 관리 여부, 할당·현재·최고 사용량과 파일을 분석합니다.
- 메모리 압축을 조회하고 일반 권한으로 확인할 수 없는 경우 조회 제한을 표시합니다.
- 프로세스별 작업 집합, 전용 커밋, 최대 작업 집합, 페이지 오류와 스레드를 수집합니다.
- Windows 구성요소를 보호하고 전용 커밋 1GB·2GB 기준으로 검토 단계를 표시합니다.
- 물리 메모리·커밋 압박과 페이지 파일 상태에 따라 정상·확인 필요·주의·위험을 판정합니다.

## 커뮤니티 벤치마크 계획

- `BENCHMARK_COMPARISON_PLAN.md`에 익명 선택 동의형 비교군을 설계했습니다.
- Geekbench 등 다른 도구의 점수는 시험 방식이 달라 CoreWatch 백분위에 혼합하지 않습니다.
- 동일 `benchmarkVersion`과 테스트 프로필 결과만 집계합니다.
- 최소 10개 표본, 중앙값·분위수, 이상치 제거와 삭제 영수증을 적용합니다.
