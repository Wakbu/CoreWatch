# CoreWatch 5.4.0 구현 상태

다른 작업에서도 이어서 개발할 수 있도록 현재 구현·검증·배포 상태를 기록합니다.

## 현재 버전

- 앱 및 설치 프로그램: `5.4.0`
- 대상: Windows x64, .NET 9 WPF
- 작업 기능: 전원·성능 설정 분석

## 5.4.0 구현 내용

- 활성 Windows 전원 계획과 AC 전원·배터리 상태를 읽습니다.
- 현재 전원 계획의 CPU AC/DC 최소·최대 상태와 프로세서 부스트 모드를 표시합니다.
- 균형 조정, 고성능, 절전 계획의 설치 여부와 현재 활성 상태를 구분합니다.
- 배터리 사용 여부와 CPU 제한 상태를 근거로 보수적인 권장 프로필을 표시합니다.
- 사용자가 확인한 경우에만 Windows 전원 계획을 변경합니다.
- 변경 직전 계획을 `%LOCALAPPDATA%\CoreWatch\PowerBackups`에 JSON으로 보관합니다.
- 마지막 백업으로 전원 계획을 복원할 수 있습니다.
- GPU 전력 정책은 제조사 드라이버 영역이므로 분석 결과에 한계를 표시하고 임의로 변경하지 않습니다.

## 안전 정책

- 분석만으로 시스템 설정을 변경하지 않습니다.
- 적용 전 확인 대화상자를 표시합니다.
- 지원하는 세 가지 Windows 기본 계획 외의 GUID는 적용 대상으로 허용하지 않습니다.
- 설치되지 않은 계획은 선택할 수 없습니다.
- 백업 GUID가 현재 Windows에 존재하는 경우에만 복원합니다.
