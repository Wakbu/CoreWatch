# CoreWatch 6.1.1 구현 상태

## 현재 버전

- 앱·설치 없는 실행본·설치 프로그램: `6.1.1`
- 이번 기능: 네트워크·업데이트·개인화·탐색·중첩 표·벤치마크 진행 UI 보정

## 구현 내용

- 네트워크 어댑터/지연 표의 좌우 경계 일치, 내부 스크롤 제거와 어댑터 8행 기본 표시·전체 펼침
- 네트워크 표 열·행 크기 변경과 열 재배치 차단
- 자동 업데이트를 OPTIMIZE 카드·기능 트리에서 제거
- 시작 후 공개 GitHub Release를 확인하고 새 버전이 있을 때만 상단 상태 옆 배지 표시
- 개인화 추천 목적을 일반·게임·콘텐츠 제작·저전력 세그먼트 버튼으로 교체
- 하드웨어 트리 선택 시 해당 탭과 실제 범주 선택을 함께 수행
- 바깥 페이지 안의 표는 내부 스크롤바 없이 기본 행만 표시하고 `모두 보기 · N개`로 전체 펼침
- 저장 공간 정리 버튼을 미리보기 표 바로 위로 이동
- 벤치마크 단계별 진행률 표시와 조회 표 선택 배경 제거
