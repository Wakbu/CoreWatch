# CoreWatch 5.5.0 구현 상태

다른 작업에서도 이어서 개발할 수 있도록 현재 구현·검증·배포 상태를 기록합니다.

## 현재 버전

- 앱 및 설치 프로그램: `5.5.0`
- 대상: Windows x64, .NET 9 WPF
- 이번 기능: 장시간 백그라운드 부하 탐지

## 5.5.0 구현 내용

- 15초, 30초, 60초 측정 시간을 선택할 수 있습니다.
- UI를 막지 않는 비동기 1초 표본화와 사용자 취소를 지원합니다.
- 프로세스별 CPU 평균·최대 사용률과 누적 CPU 시간을 계산합니다.
- 메모리 평균·최대 작업 집합을 기록합니다.
- Windows `GetProcessIoCounters`로 디스크 읽기·쓰기 누적량과 평균 처리량을 계산합니다.
- IPv4·IPv6 TCP 테이블에서 활성 네트워크 연결의 평균·최대 개수를 기록합니다.
- 앱, 백그라운드, Windows 프로세스를 구분합니다.
- 지속 부하 높음, 검토 권장, 정상 범위, Windows 구성요소, 측정 도구로 판정합니다.
- Windows 구성요소에는 종료 금지 안내를 제공합니다.
- PID 기반으로 기존 행을 갱신하고 순위만 이동해 측정 중 표 깜빡임을 줄였습니다.

네트워크 값은 프로세스별 전송 바이트가 아니라 관리자 권한 없이 확인 가능한 활성 TCP 연결 개수입니다.
