# CoreWatch 시스템 최적화 기능 로드맵

## 5.3.0 구현 완료

- 시작 프로그램 분석: 활성 등록 항목, 게시자, 등록 위치, 영향과 보수적인 권장 사항
- 저장 공간 분석: 사용자/Windows 임시 파일, 충돌 덤프, 웹 캐시의 예상 확보 용량
- 사용자 승인형 정리: 항목 선택, 예상 용량 확인, 최종 확인 후 24시간 이상 된 파일만 정리
- 최적화 전후 비교: CPU, 메모리, 프로세스 수, 디스크 최소 여유 공간 기준 기록과 변화 표시
- 모든 분석과 파일 검색을 백그라운드에서 수행

- ✅ 시작 프로그램 활성화·비활성화, 보호 정책, 변경 전 백업·되돌리기 (5.6.0 완료)
- ✅ 메모리 분석: 물리·커밋·캐시·커널 풀, 페이지 파일, 압축과 프로세스 전용 커밋 (5.7.0 완료)

- GitHub Release 기반 자동 업데이트와 SHA-256 검증
- Windows 저장소 센스와 예약 최적화 설정

- PC 구성과 사용 목적을 반영한 개인화 추천 점수

## 안전 원칙

- 모든 변경은 미리보기와 사용자 확인 후 실행합니다.
- 변경 전 현재 설정을 백업하고 가능한 작업은 되돌리기를 제공합니다.
- 관리자 권한은 실제 시스템 변경 시점에만 요청합니다.
- Windows 핵심 서비스, 보안 기능과 업데이트를 무조건 비활성화하지 않습니다.
- 근거 없는 레지스트리 트윅, RAM 클리너와 SSD 강제 조각 모음은 제공하지 않습니다.
- 삭제 작업은 정확한 대상과 예상 확보 용량을 먼저 표시합니다.
- 최적화 전후 지표를 측정해 실제 효과가 없는 변경을 구분합니다.
